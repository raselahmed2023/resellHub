# ReSell Hub

## Project Purpose
ReSell Hub is a second-hand marketplace platform where users can buy and sell pre-owned products safely and efficiently. It helps reduce waste, promote sustainable consumption, and create opportunities for users to earn money from unused items.

## Key Features
-  Authentication (Email/Password + Google) via Better Auth
-  Role-based dashboard (Buyer, Seller, Admin)
-  Product listing with image upload (ImgBB)
-  Admin product approval system
-  Stripe payment integration
-  Wishlist management
-  Sales analytics with charts
-  Advanced search, filter & sort
-  Pagination
-  Sustainability impact section
-  Framer Motion animations
-  Fully responsive design
